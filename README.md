## Ubicación de carpetas

### Scripts

La carpeta `Scripts` debe ubicarse en la siguiente ruta:

```bash
/home/user/Scripts
```

Donde `user` debe reemplazarse por el nombre de tu usuario de Linux correspondiente.

Por ejemplo:

```bash
/home/santi-kun/Scripts
```

### ApplicationsDesktop

La carpeta `applications` contiene los archivos lanzadores de aplicaciones para Linux.

Estos archivos deben colocarse en la siguiente ruta:

```bash
/home/user/.local/share/applications
```

Esta carpeta ya existe en Linux, copia el contenido de los archivos ahí.
La misma es utilizada por Linux para guardar los lanzadores de aplicaciones del usuario.  
Los archivos `.desktop` que se coloquen allí pueden aparecer en el menú de aplicaciones del sistema.
